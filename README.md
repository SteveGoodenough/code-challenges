# code-challenges

## Setup your python environnment

Use `pipenv` !! https://pipenv.readthedocs.io/en/latest/

## Setup the project

`pipenv install --dev`

## Run the tests with coverage

Anything less than 100% will fail

`pytest --cov=src`

## Lint the project

`flake8`

## Run continual testing

`ptw`
